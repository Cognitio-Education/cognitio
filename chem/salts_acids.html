Salts and Acids

Revision Notes

We can produce salts by reacting certain elements and compounds with acid.

Salts from Metals

When a metal is added to an acid, a reaction usually takes place.

As a general rule, the equation for this reaction is:

acid + metal -> salt + hydrogen

As we know, a salt is formed when an acid and an alkali are formed, and therefore we can infer that metals are alkaline.

The salt that is formed is dictated by the type of acid that is reacted, and NOT the metal. The three main acids we will look at are hydrochloric acid (HCl), nitric acid (HNO3), and sulfuric acid (H2SO4). The resulting salts are as follows:

metal + hydrochloric acid -> metal chloride + hydrogen

metal + nitric acid -> metal nitrate + hydrogen

metal + sulfuric acid -> metal sulfate + hydrogen

For example, if we reacted magnesium and hydrochloric acid, we would be left with some magnesium chloride (MgCl), and additionally hydrogen gas. We can check for this hydrogen gas using the lit splint-squeaky pop test.

However, not all metals can react with acid. If we try to react copper with hydrochloric acid, nothing happens, and the lit splint-squeaky pop test does not work. This is because copper is not reactive enough to react with acids, and this also applies to elements less reactive than copper, such as gold and silver.

Salts from Bases

Another way of obtaining a salt is from bases/alkalis (remember that an alkali is a water-soluble base), such as metal hydroxides (which can be made using metals and water, metal oxides (formed when metals react with oxygen) and metal carbonates (formed when metal oxides react with carbon dioxide). The results of these compounds reacting with acid are as follows:

metal oxide + acid -> salt + water

metal hydroxide + acid -> salt + water

metal carbonate + acid -> salt + water + carbon dioxide

The rules for what salt is formed are the same as the ones for metal elements + acid (see "Salts from Metals"). For example, if we react sodium carbonate (Na2CO3) with hydrochloric acid, we will be left with sodium chloride, water, and carbon dioxide.

One key reason why being able to form salts from bases is so important is because it allows us to make salts out of otherwise unreactive elements. For example, copper sulfate cannot be obtained from copper in element form, as it is not reactive enough. However, if we react copper hydroxide with sulfuric acid, then we can obtain copper sulfate.

Ammonia and Acid

The reaction of acid with ammonia (NH3) is another important one which you are required to know. When reacted with acid, ammonia produces an ammonium salt. As with metals, this salt depends on the acid used. The equation is as follows:

ammonia + acid -> ammonium salt

For example, reacting ammonia with hydrochloric acid produces ammonium chloride (NH4Cl). Nothing else is produced in this reaction.
